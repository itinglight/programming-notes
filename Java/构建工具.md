### Gradle项目目录结构介绍

```
src/main/java 放置正式代码的目录
src/main/resouces 放置正式配置文件目录
src/test/java 放置单元测试代码目录
src/test/resources 放置测试配置文件目录
src/main/webapp 放置页面元素，比如 js,css,jsp,html 等等
```
