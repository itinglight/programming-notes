# Linux

### Linux的安装

### 快捷键：

| 快捷键              | ---              |
| ------------------- | ---------------- |
| Tab                 | 连按两下补充命令 |
| [Ctrl + c ]         | 中断目前程序     |
| [Ctrl +d ]          | (相当于输入quit) |
| [Shift + Page Up]   | 向前翻页         |
| [Shift + Page Down] | 向后翻页         |
| q                   | 离开当前文档     |

| ps -aux    | 查看后台执行的程序   |
| shutdown   | 关机                 |
| sync       | 将数据同步写入硬盘中 |
| reboot     | 重新启动             |
