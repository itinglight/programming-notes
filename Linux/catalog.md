# Linux

### Linux的安装

### 快捷键：

| 快捷键              |                  |
| ------------------- | ---------------- |
| Tab                 | 连按两下补充命令 |
| [Ctrl + c ]         | 中断目前程序     |
| [Ctrl +d ]          | (相当于输入quit) |
| [Shift + Page Up]   | 向前翻页         |
| [Shift + Page Down] | 向后翻页         |
| q                   | 离开当前文档     |

###  windowsCMD常用命令：
|				|				|
| ---- | ---- |
| dir  | 列出当前目录下的文件以及文件夹 |
| md   | 创建目录 |
| rd   | 删除目录 |
| cd   | 进入指定目录 |
| cd.. | 退回到上一级目录 |
| cd\  | 退回到跟目录 |
| del | 删除文件 |
| exit | 退出dos命令行 |
|      |      |
|      |      |

| shutdown   | 关机                           |

| reboot     | 重新启动                       |
| halt       |                                |
| poweroff   |                                |
| pwd        | 获取目前所在工作目录的绝对路径 |
| Cat        | 显示文件内容                   |
